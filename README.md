## 📋 如何启用GitHub Pages

如果网站无法访问，请按照以下步骤手动启用：

### 方法1：通过GitHub界面（推荐）

1. 访问你的仓库：https://github.com/xianyu110/gemini3flash
2. 点击 **Settings** 标签页
3. 在左侧菜单中找到 **Pages**
4. 在 "Source" 下选择 **Deploy from a branch**
5. 设置：
   - Branch: **main**
   - Folder: **/(root)**
6. 点击 **Save**

### 方法2：等待GitHub Actions自动部署

如果已经配置了GitHub Actions，等待几分钟让自动部署完成。
